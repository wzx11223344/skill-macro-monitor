# 宏观经济监测 (Macro Economy Monitor)

自动抓取中国宏观经济 9 大核心指标，生成日报/周报/月报。
所有数据通过 akshare 连接国家统计局、中国人民银行、海关总署等官方数据源。

## 能力边界

### ✅ 可以做什么
- 自动获取中国宏观经济核心指标（PMI、CPI/PPI、M2、社融、进出口、工业增加值、固投、社零）
- 生成三种频率的报告：日报（核心指标一览）、周报（趋势+预警）、月报（图文深度解读）
- 自动计算趋势方向、历史分位、交叉信号预警
- 输出 Markdown（终端）或 HTML（带Plotly交互图表的浏览器报告）

### ❌ 不可以做什么
- **不覆盖非中国宏观经济数据**（如美股、外汇、加密货币等）
- **不提供实时高频数据**（数据频率为月度，有1-2个月的发布延迟）
- **不包含行业/个股/基金分析**（这是宏观工具，不是金融分析工具）
- **不做预测/量化模型**（只做描述性监测，不做预测）
- **不支持自定义指标**（指标列表是固定的 9 项）
- **数据源依赖 akshare 和网络连接**（如果 akshare 接口变更或网络不通，可能暂时无法获取部分数据）

## 使用方法

```bash
# 日报模式：当日核心指标 + 一句话总结
python monitor.py brief

# 周报模式：趋势分析 + 交叉信号 + 预警
python monitor.py weekly

# 月报模式：全面文字解读 + 9张Plotly交互图表 + HTML报告
python monitor.py monthly

# 指定输出路径
python monitor.py monthly --output my_report.html
```

## 常见问题 (FAQ)

### Q: 运行报错「akshare 模块未找到」怎么办？
A: 运行 `pip install -r requirements.txt` 安装依赖。

### Q: 某些指标显示「暂无数据」怎么办？
A: 这是正常的。原因可能是：
1. 该指标当月数据尚未发布（有1-2个月延迟）
2. 网络问题导致获取失败（工具会自动重试3次）
3. 极少情况下 akshare 接口变更
建议：稍后重试，或检查网络连接。如果持续失败，可到 akshare 官方文档确认接口是否变更。

### Q: 为什么数据看起来比较旧（比如显示上个月的数据）？
A: 宏观经济数据通常有 1-2 个月的发布延迟。例如 3 月发布的 PMI 是 2 月的数据。这是正常现象。

### Q: 生成的图表为什么看不到？
A: 月报 HTML 中的图表依赖 CDN 加载 Plotly.js。需要联网才能看到图表。如果是离线环境，请确保已安装 plotly 并且浏览器有缓存。

### Q: 这个工具和 Wind/Bloomberg 比有什么不同？
A: 完全开源免费，数据来自公开渠道（国家统计局等），不需要付费终端。但数据覆盖范围和实时性不如专业金融终端。

### Q: 交叉信号是怎么算的？准确吗？
A: 交叉信号基于经验规则（如"PMI下行 + PPI下行 = 需求不足"），不是量化模型。可以作为参考，但不构成投资建议。

### Q: 可以自定义添加指标吗？
A: 目前不支持。9 项指标是预设的。如需扩展，可以修改 `FETCHERS` 列表和对应的 `fetch_xxx()` 函数。

## 技术栈
- 数据源: akshare → 国家统计局 / 中国人民银行 / 海关总署
- 数值: numpy / pandas
- 图表: plotly (交互式)
- 缓存: JSON + 24h TTL
- 报告: 纯 Python 生成 HTML
